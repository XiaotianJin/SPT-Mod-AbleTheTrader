# MOD 简述｜Project Genesis-Last
> 在塔科夫，死亡不是失败。
> 无法被重复，才是。

本 MOD 引入了一个隐藏于战火阴影中的全新商人——Abel，
以及其背后那家从未被正式记录的地下组织：
- 圣仪系统公司（Liturgy Systems）

他们相信一件事：

> 如果奇迹可以被流程化、被复制、被量产，
> 那它就不再是神话，而是技术。

## 核心内容

### 🧩 完整任务线商人 Abel

- 从黑市传闻中登场

- 以“委托”的形式逐步引导玩家接触其产品

- 不强迫信仰，只展示结果

### 🕯️ 四阶段产品体系

- 正典计划（Porject Canon）：略优于制式的可靠改良

- 礼赞协议（Project Laudation）：接近顶级的工艺极限

- 代行者计划（Project Executor）：超越现有技术的全新尝试

- 创世-终章计划（Porject Genesis-Last）：接近神的领域才能获取的恩赐

随着任务的推进，四大系列会逐渐解锁，你将会在Mod中体会到：

>💉 超规格医疗、兴奋剂与状态协议：
>修复、抑制、强化、透支，
>每一次存活，都伴随着代价。

>🛡️ 实验级防护终端与战术组件：
>从原型机到“现象级”装备，
>不断突破塔科夫的既有数值边界，又维持着微妙的平衡。

>📦 安全箱与携行系统的彻底重构：
>“越早拿到3x4，游戏难度越低，但是越早想拿3x4，就越难，所以游戏难度越低，游戏难度越高”——不知名游戏玩家。
>从正典扩容安全箱，
>到代行者的 “珍奇柜”，
>再到创世阶段的 “诺亚方舟” 与 “盗火者”，
>安全箱大小不再是限制，而是你可选的对象。

>🔥 IMBA的特殊弹药：
> 点杀伤、面杀伤、超规格的裁决和审判
